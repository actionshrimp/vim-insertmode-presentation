Some things you can do in insert mode...

VimLondon, 28th May 2013

Getting out of insert mode

Esc, C-[, C-c

Ctrl+[

Ctrl+c - doesn't work with abbreviations and doesn't trigger InsertLeave

Possible to map C-c to Esc to get full behavior - perhaps has side effects though. Also bad for your hands!

Basics

C-w (think word)

C-u (think uh-oh)

C-j (think hjkl)

C-h (think hjkl)

C-i (think indent)

C-a

Bonus: Top three also work in the shell (in both vi and emacs mode)!

Registers

C-r register

expression register

Registers: + expression register: ++ current date: "=strftime('%c') ++ vim value, e.g. "=&guioptions + : or / registers, e.g. ":, "/ (although better done with q: and q/)

Text width

Most useful for actual 'text' editing rather than coding

gw, gwgw/gww, ^ogwap

Abbreviations

Auto-fix common typos: teh -> the

Completion

C-p / C-n - bread and butter completion

C-l - line completion, suprisingly handy, smart about tabs

Getting help for insert mode

prefix the thing you're looking for with 'i_'

:help i_^w

(this also works for other modes, e.g. v_ for visual)

also... :help Insert-mode
